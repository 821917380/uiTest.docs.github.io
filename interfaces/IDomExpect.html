    Interface IDomExpect<T>

    IDomExpect 接口继承自 IBaseExpect，用于对 DOM 元素进行断言。 该接口提供了 DOM 元素特有的断言方法，如可见性检查、空元素检查等。 通过实现此接口，可以创建 DOM 元素断言类，扩展基本的断言功能。

    not

    not: IBaseExpect<T>

    反转断言逻辑，使得后续的断言变为其相反的结果。

    Returns

    当前的断言对象，便于链式调用。

    Example

    expect(5).not.toBe(3); // 断言 5 不等于 3
    expect('hello').not.toBe('world'); // 断言 'hello' 不等于 'world'
    expect(null).not.toBeDefined(); // 断言 null 未定义
    expect(undefined).not.toBeNull(); // 断言 undefined 不为 null
    expect(NaN).not.toBe(NaN); // 断言 NaN 不等于 NaN
    expect(document.querySelector('.btn-primary')).not.toBeNull(); // 断言元素存在
